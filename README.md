Student Performance Analysis Using Data Analytics

1\. Project Overview



This project focuses on analyzing student academic performance using data analytics techniques.

The objective is to understand the factors that influence student outcomes such as Dropout, Enrolled, and Graduate by analyzing demographic, academic, and financial attributes.



2\. Objectives



* Analyze student demographic and academic data
* 
* Identify factors contributing to student dropout and graduation
* 
* Perform exploratory data analysis (EDA)
* 
* Visualize patterns and trends in student performance
* 
* Extract meaningful insights to support academic decision-making
